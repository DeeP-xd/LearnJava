Operacje na plikach

Zapis i odczyt plików w Javie jest stosunkowo prosty. Wystarczy poznać podstawy i nie przestraszyć się kilku nowych klas.

Odczyt

Pierwszą klasą, która pozwoli nam utworzyć obiekt przechowujący dane pliku to File:

File file = new File(“plik.txt”);

Utworzyliśmy nowy obiekt typu File o nazwie “file”, do którego przypisaliśmy plik o nazwie “plik.txt”. Najprostszym sposobem na odczyt danych z pliku jest użycie klasy Scanner.

Scanner scanner = new Scanner(file);

Obie linie możemy połączyć, unikając zbędnego tworzenia obiektu File:

Scanner scanner = new Scanner(new File(“plik.txt));

Plik możemy odczytać linia po linii używając:

String line = scanner.nextLine();

Cały program (po zaimportowaniu odpowiednich klas) powinien wyglądać następująco:

import java.io.File;
import java.io.FileNotFoundException;
import java.util.Scanner;

public class Read {
public static void main(String[] args) throws FileNotFoundException {
File file = new File(“plik.txt");
Scanner scanner = new Scanner(file);

String line = scanner.nextLine();
System.out.println(line);
}
}

Aby program zadziałał, plik, który importujemy powinien znaleźć się w tym samym folderze, co plik .class, w innym przypadku podajemy pełną ścieżkę do pliku (np. “/Users/user/plik.txt”).

Jeżeli chcemy odczytać cały plik najprostszym sposobem jest użycie pętli while.

import java.io.File;
import java.io.FileNotFoundException;
import java.util.Scanner;

public class Read {
public static void main(String[] args) throws FileNotFoundException {
File file = new File(“plik.txt");
Scanner scanner = new Scanner(file);

String line = scanner.nextLine();

while(line != null) {
System.out.println(line);
line = scanner.nextLine();
}
}
}
