Zapis

Do zapisu do pliku używamy klasy PrintWriter, której parametrem jest nazwa pliku / ścieżka do pliku, do którego chcemy zapisać dane.

PrintWriter writer = new PrintWriter(“plik.txt”);

Tworzymy obiekt writer typu PrintWriter, do którego bezpośrednio przypisujemy plik (bez tworzenia innego obiektu, ani użycia klasy File). Kompletny program zapisujący dane wygląda w ten sposób:

import java.io.FileNotFoundException;
import java.io.PrintWriter;

public class Write{
public static void main(String[] args) throws FileNotFoundException{
PrintWriter writer = new PrintWriter(“plik.txt");
writer.println("Ala ma kota, a kot ma Alę");
writer.close();
}
}

Mimo, że Java posiada mechanizmy zwalniania pamięci, zamknięcie strumienia (writer.close();) jest bardzo ważne.
