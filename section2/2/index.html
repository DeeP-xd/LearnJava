Operatory

  • + - dodawanie
  • - - odejmowanie
  • * - mnożenie
  • / - dzielenie
  • % - modulo (reszta z dzielenia)
  • + - dodawanie
  • - - odejmowanie
  • * - mnożenie
  • / - dzielenie
  • % - modulo (reszta z dzielenia)
  • ++ - dodaje 1 do wartości zmiennej
  • -- - odejmuje 1 od wartości zmiennej
  • += - dodaje wartość do zmiennej przed znakiem
  • -= - odejmuje wartość od zmiennej przed znakiem
  • *= - mnoży zmienną przed znakiem przez wartość
  • /= - dzieli zmienną przed znakiem przez wartość
  • %= - przypisuje do zmiennej resztę z dzielenia jej przez wartość

Przykłady:

int a = 3 + 2; // a = 5
int b = 12 * 3; // b = 36
double c = 5 / 2; // c = 2.5
int d = 15 / 4; // d = 3

a++; // a = 6
b--; // b = 35

a += 4; // a = 10
b -= 12; // b = 23

a /= 5; // a = 2

b %= 5; // b = 3
