switch

Instrukcja warunkowa, którą najprościej wyobrazić sobie jako menu - wybieramy jedną opcję spośród dostępnych i przechodzimy dalej.

int x = 5;
String day;

switch (x) {
case 1:
day = “Poniedziałek”;
break;
case 2:
day = “Wtorek”;
break;
case 3:
day = “Środa”;
break;
case 4:
day = “Czwartek”;
break;
case 5:
day = “Piątek”;
break;
case 6:
day = “Sobota”;
break;
case 7:
day = “Niedziela”;
break;
default:
day = “błąd”;
break;
}

// day = “Piątek”

W tym przykładzie mamy 2 zmienne: “x", który oznacza numer dnia tygodnia oraz zmienną “day”, która przyjmuje odpowiednią wartość (nazwę dnia) do wartości x. W tym przypadku będzie to piątek.
