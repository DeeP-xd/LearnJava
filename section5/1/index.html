Zmienne łańcuchowe

Czyli łańcuchy znaków, które najprościej nazwać tekstem. Typem danych, którym się zajmiemy jest String, czyli ciąg dowolnych znaków, albo inaczej tablica zmiennych typu char. Możemy w zmiennej typu String zapisać słowo, zdanie lub dowolne znaki (lecz nie więcej niż 2 147 483 647). W praktyce wygląda to w ten sposób:

String name = “Bartek”;
String sentence = “Ala ma kota.”;
String postCode = “12-220”;

Jak wcześniej wspomniałem zmienna typu String w praktyce jest tablicą pojedynczych znaków, dlatego możemy odwołać się do konkretnego znaku, jak do elementu tablicy:

char x = name[0];

// x = ‘B’

Przykład praktycznego użycia tej właściwości:

String name = “Katarzyna”;
bool isMale;

if (name[name.length - 1] == ‘a’) {
isMale = false;
} else {
isMale = true;
}

Program sprawdza czy ostatni znak w imieniu, to “a” (wszystkie żeńskie imiona kończą się na “a”). Odwołując się do ostatniego elementu, a nie znając długości łańcucha znaków (imiona mogą mieć różną długość, dlatego nasz kod powinien być uniwersalny), najlepiej użyć zapisu name[name.length - 1]. Dlaczego “-1”? Ponieważ name.length zwraca nam rzeczywistą długość ciągu (w tym przypadku 9), a poszczególne znaki są indeksowane od 0 (patrz: Tablice). Dlatego ostatni znak (w tym przypadku “a”) ma indeks 8, dlatego chcąc się odwołać do ostatniego znaku, musimy od długości ciągu odjąć 1. Zmienna isMale przyjmie wartość true, jeżeli imię będzie męskie (nie będzie kończyło się na “a”), lub wartość false, jeżeli imię będzie żeńskie (będzie kończyło się na “a).
