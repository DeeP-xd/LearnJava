Klasy

Klasa, czyli “szablon” dla obiektu, który chcemy utworzyć składa się z kilku elementów:

  • nazwy,
  • atrybutów,
  • metod.

Zaczynając od początku, nazwa klasy powinna określać jaką “foremkę” chcemy stworzyć, np. samochód.

class Car {

}

W taki sposób utworzyliśmy klasę o nazwie “Car” (ze względu na dobre praktyki, warto od początku używać angielskiego, nazywając klasy, zmienne, itp).

Skoro mamy już “foremkę” do naszego samochodu, zastanówmy się jakie atrybuty może mieć samochód.

class Car {
String brand;
String model;
int year;
int mileage;
double fuelLevel;
}

W tym momencie nasz samochód posiada 4 atrybuty: markę, model, rocznik, przebieg oraz poziom paliwa. Kolejnym etapem będzie dodanie kilku metod (funkcji) do naszego samochodu.

class Car {
String brand;
String model;
int year;
int mileage;
double fuelLevel;

public void drive (speed) {
// tu opisujemy co dokładnie nasz pojazd ma zrobić, żeby jechać
}

public void stop () {
// tu opisujemy co dokładnie nasz pojazd ma zrobić, żeby się zatrzymać
}

public String getInfo () {
return mileage + “ “ + fuelLevel;
}
}

Mamy 2 metody typu void (które nie zwracają żadnej wartości) i jedną typu String, która zwraca nam ciąg znaków (w postaci przebiegu oraz poziomu paliwa, oddzielone spacją).
