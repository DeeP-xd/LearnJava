Teraz możemy dodać trochę funkcjonalności. Dla przykładu, kiedy używamy samochodu zużywamy paliwo, dlatego:

class Car {
String brand;
String model;
int year;
int mileage;
double fuelLevel;

public void drive (speed) {
// tu opisujemy co dokładnie nasz pojazd ma zrobić, żeby jechać
fuelLevel -= 0.2;
}

public void stop () {
// tu opisujemy co dokładnie nasz pojazd ma zrobić, żeby się zatrzymać
}

public String getInfo () {
return mileage + “ “ + fuelLevel;
}
}

Teraz za każdym razem, kiedy użyjemy metody “drive”, z naszego samochodu ubędzie trochę paliwa. Na takiej zasadzie możemy zmieniać wartość różnych zmiennych. A co jeżeli będziemy chceli przekazać do funkcji jakąś wartość? Może czas zatankować samochód.

class Car {
String brand;
String model;
int year;
int mileage;
double fuelLevel;

public void drive (speed) {
// tu opisujemy co dokładnie nasz pojazd ma zrobić, żeby jechać
fuelLevel -= 0.2;
}

public void stop () {
// tu opisujemy co dokładnie nasz pojazd ma zrobić, żeby się zatrzymać
}

public String getInfo () {
return mileage + “ “ + fuelLevel;
}

public void tank (double fuel) {
fuelLevel += fuel;
}
}

W tym momencie po tankowaniu (tank) naszego samochodu do aktualnej ilości paliwa, którą mieliśmy wcześniej (fuelLevel) dodajemy ilość zatankowanego paliwa (fuel).
